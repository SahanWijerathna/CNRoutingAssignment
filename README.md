## Router Configuration and Routing Protocols Assignment

This repository contains the Cisco Packet Tracer files for the Computer Networks assignment.

### Files Included:
1. **Static_Routing.pkt** - Implementation of static routes across three networks.
2. **RIP_Routing.pkt** - Dynamic routing implementation using RIP version 2.
3. **EIGRP_Routing.pkt** - Dynamic routing implementation using EIGRP (AS 10) after removing RIP configuration.
